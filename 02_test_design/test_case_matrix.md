# 테스트케이스 매트릭스

## 1. 문서 목적

본 문서는 PC방 매장 운영 플랫폼의 주요 테스트케이스를 정의한다.

테스트케이스는 요구사항, 리스크 분석, 핵심 시나리오를 기반으로 작성되었으며, 이후 AI 테스트케이스 생성 결과 검토 및 Playwright 자동화 대상 선정의 기준으로 사용된다.

---

## 2. 테스트케이스 작성 기준

| 항목        | 설명                       |
| ----------- | -------------------------- |
| TC ID       | 테스트케이스 고유 ID       |
| 기능        | 테스트 대상 기능           |
| 테스트 목적 | 검증하려는 목적            |
| 사전 조건   | 테스트 수행 전 필요한 조건 |
| 테스트 단계 | 테스트 수행 절차           |
| 기대 결과   | 정상적으로 기대되는 결과   |
| 우선순위    | High / Medium / Low        |
| 테스트 유형 | E2E / API / Manual         |
| 관련 리스크 | 2단계 리스크 분석 ID       |
